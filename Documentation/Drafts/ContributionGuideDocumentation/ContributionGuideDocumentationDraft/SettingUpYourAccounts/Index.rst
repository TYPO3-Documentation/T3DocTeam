==============================
Setting up your Github account
==============================

In order to be able to make changes to the documentation, add issues 
etc. you need a Github account. Follow one of the many resources or
just go to https://github.com and fill out the form "Sign up for Github".

You will need to provide an email adress that you actually use, because 
you will get notifications about comments to your changes.

.. hint::
   Use the same email-adress you use for your typo3.org and Slack account.
